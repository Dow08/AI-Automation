# AI & Automation

Projets autour des agents IA locaux, de l'automatisation et de la surveillance reseau.
Approche orientee outils pratiques et deploiement offline.

## Projets

### agent-hermes-local
Agent IA personnalise tournant 100% en local — multi-modeles (Ollama, Mistral, GLM), dashboard adaptatif, selection dynamique du LLM selon la tache.
Stack : Python, Ollama, Mistral, GLM

### actu-cybersecurite
Scraper de veille cybersecurite — agregation automatique de news et podcasts depuis plusieurs sources, classement par thematique.
Stack : Python

### netguard-surveillance
Backend de surveillance reseau — monitoring des connexions et detection d'anomalies.
Stack : Python
