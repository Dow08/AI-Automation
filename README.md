# AI & Automation

Projets autour des agents IA locaux, de l'automatisation et de la surveillance réseau.
Approche orientée outils pratiques et déploiement offline.

## Projets

### agent-hermes-local
Agent IA personnalisé tournant 100% en local — multi-modèles (Ollama, Mistral, GLM), dashboard adaptatif, sélection dynamique du LLM selon la tâche.
Stack : Python, Ollama, Mistral, GLM

### actu-cybersecurite
Scraper de veille cybersécurité — agrégation automatique de news et podcasts depuis plusieurs sources, classement par thématique.
Stack : Python

### netguard-surveillance
Backend de surveillance réseau — monitoring des connexions et détection d'anomalies.
Stack : Python
