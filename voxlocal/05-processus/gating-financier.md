# Gating financier 0-10 (agent LEDGER)

Aucun euro engagé sans ROI prouvé.

| Score | Signification | Action |
| :-: | :-- | :-- |
| **0-5** | Rentabilité non prouvée | ❌ Zéro euro · alternative open source obligatoire |
| **6-8** | Rentabilité possible, perf faible | ⏸️ Rester gratuit, optimiser, re-tester |
| **9-10** | ROI sûr et quantifié | ✅ Dépense autorisée |

**Exemple :** on ne paie un numéro/minute Vapi que quand un prospect a dit oui (ROI quasi-certain = 9-10). Tant qu'on prospecte → free tier.

**Coût de build = ≈ 0 €** (outils OSS + free tiers). Seuls les COGS (voix/téléphonie) sont engagés, et seulement après vente.
